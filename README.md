Name of the app: How you doin'? Description of app: This app elaborates on the phenomena of mental illness and helps those suffering from symptoms find a good help source to ease their struggles. Through videos and references, we help them get one step closer to their desired head space.

DISCLAIMER: We are not an app for psychological counselling or any related medical service.

Technologies: Node.js (Express), Javascript (Axios), HTML5, CSS, PostgresSQL, Heroku (host for backend), Github pages (host for frontend)
